ゲーム概要

数学の知識を武器に戦う魔道士のローグライクRPG。敵の頭上に表示された数値をぴったり0にして倒す。宝箱から知識を獲得し、技やフィールドアクションとして活用しよう。

戦闘の基本

  • 敵の数値を技で変化させ、ぴったり0にすると撃破
  • 数値が負になると幽霊化（壁通過、ダメージ無効、攻撃でのみ数値変動）
  • 幽霊も技で0にしないと倒せない
  • ぴったり0で倒すとコンボ加算、幽霊化させるとコンボリセット

範囲攻撃（関数グラフ技）Coming Soon

数学の関数をフィールド上にグラフとして描画し、軌道上の全敵に同時ヒットする範囲攻撃。
技キーを押すと攻撃範囲のプレビューが表示され、向きを調整してから発動できる。隣接不要。

コンボ戦略

範囲 + 単体のコンボ例:
  • 削り→仕留め: 一次関数(-5)で通路の敵3体を一度に削り、個別に-1で0に調整してコンボ撃破
  • 円 + モジュロ: 円の方程式(-3)で周囲を削り、%4で4の倍数の敵を即0化
  • 指数 + 対数: 指数関数で遠方の敵を大幅に削り、log2で一桁に → -1で仕留め
  • 波形で掃討: 三角関数で大部屋を薙ぎ払い、残った敵を通常技で仕留める
注意: 範囲攻撃で複数敵を同時に負の数にすると大量の幽霊が発生してコンボリセット。削りは慎重に、仕留めは個別で。

攻略Tips

序盤 (1F-5F): 技なしでスタート。開始部屋の宝箱で「減法(-1)」を入手。子狼(1-2)は-1で1-2ターン。宝箱で「繰り下がり(-5)」が出たらラッキー! 熊(4-8)は無理せず回避してもOK。
中盤 (6F-15F): /2(ハーフ)が手に入ると世界が変わる。大きい数値の敵は /2 を連打して小さくしてから -1 で仕留める。%4(モジュロ)は4の倍数の敵に即効。偶数の粉 + /2 のコンボも強力。ステージが変わるとマップの見た目も変化!
終盤 (16F-25F): derivative(桁数変換)とintegral(各桁合計)が強い。100 -> derivative -> 3 -> -1x3 で撃破。identity(n/n=1)は全ての敵を1にできる最高の汎用技。ゼロ乗算(x0)はボス戦に温存（全ゲーム通して3回限定）。
幽霊化に注意: 数値を負にすると幽霊化。幽霊は壁を通過してくるので厄介。abs()で正に戻してから処理するか、+系の技で0に調整する。幽霊化させるとコンボがリセットされるのでスコアにも影響。
偽アイテムに注意: ステージ3以降、薬草や知恵の水に見せかけた偽物が出現。使うまで見分けがつかない。HPに余裕がある時に試すのが安全。
敵との初遭遇: 敵と隣接した最初のターンは攻撃されない。落ち着いて向きを合わせてから技を使おう。
ボス攻略の定石:
  • 原始の王(20): -5 x2、または -1 x10

  • 計算機械(100): derivative -> 3 -> -1 x3、または /2 連打
